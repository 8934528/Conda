# Cobra IDE - Modern Game Development Suite

Cobra is a high-performance, integrated development environment designed specifically for Python game creators. It combines the power of a professional code editor with the ease of use of a visual scene designer.

## Documentation

Explore the project documentation to learn more about Cobra:

- **[IDE Overview](Docs/ide_overview.md)**: Features, vision, and core capabilities.
- **[User Guide](Docs/user_guide.md)**: A step-by-step guide to creating and running games.
- **[System Architecture](Docs/architecture.md)**: Deep dive into the technical design, ECS, and UI framework.

## Key Features

- **Integrated Monaco Editor**: Professional-grade coding experience.
- **Visual Scene Designer**: Drag-and-drop game objects and sprites.
- **Python Integration**: Built-in support for venv and pip management.
- **Unified Settings**: Configure everything from editor fonts to game physics in one place.
- **Customizable UI**: High-quality dark and light themes with modern aesthetics.

## Getting Started

### Prerequisites

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [Python 3.x](https://www.python.org/)

### Installation

1. Clone the repository.
2. Open the solution in Visual Studio 2022 or use the CLI:

   ```bash
   cd Cobra
   dotnet run
   ```
